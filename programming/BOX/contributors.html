Contributors

Mark Volkmann started the BOX project, did the initial design work, and has written all the code so far. Mark works for Object Computing, Inc. in St. Louis where he provides OO, Java and XML consulting. He also creates and teaches courses in Java and XML. Mark can be contacted at mark@ociweb.com.

Brian Gilstrap, also of OCI, contributed significant design improvements. Brian provides OO, Java, CORBA and XML consulting. He also creates and teaches courses in Java. He's the author of Lumberjack, an implementation of JSR47 (the Java Logging API). Brian can be contacted at brian@gilstrap.com. Thanks Brian!

Jeff Brown, also of OCI, contributed by testing the impact of zipping BOX. While it turned out to be slower doing this than sending unzipped BOX, it was quite valuable to confirm this. This test is described in more detail in the FAQ. In addition to contributing this test, Jeff also identified several deficiencies in the BOX documentation. Jeff teaches numerous Java courses, develops courseware and provides Java consulting. He's the author of JarSpy. Jeff can be contacted at brown_j@ociweb.com. Thanks Jeff!
