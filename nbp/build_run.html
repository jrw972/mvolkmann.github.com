Building Demo App.

Steps to Build

The steps to build dsinfra and the included applications under Windows are as follows:

  1. get the file Landmark-dsinfr-win32.xml and copy it to C:\Documents and Settings\{username}\.ivy2\cache\dsinfra-LOCAL
  2. open a Command Prompt
  3. cd to the dsinfra directory (mine is C:\Halliburton\infra)
  4. run "lgcbuild module"
  5. select release specification 3

Steps to Run

The steps to run the "TabView Example 2" demo application under Windows are as follows:

  1. cd to the dsinfra directory (mine is C:\Halliburton\infra)
  2. run "lgcbuild start"
  3. select release specification 3
  4. the "Decision Space" launcher will be displayed
  5. select "TabView Example 2"

Steps to create a NetBeans project for dsinfra

  1. delete the build and nb subdirectories
  2. press the "New Project" toolbar button
  3. enter "dsinfra" for the project name
  4. select Java...Java Project with Existing Sources
  5. press "Next >"
  6. press "Finish"
  7. in the Explorer window, right click the dsinfra project and select Properties
  8. select the "Libraries" category
  9. press the "Add Library..." button and add the "JUnit 4.1" library
  10. press the "Add Library..." button and add the "JAXB 2.1" library
  11. press the "Add JAR" button and add the ant.jar file
  12. press the "Add JAR" button and add the ant-junit.jar file
  13. press the "Add JAR" button and add the com_lgc_annotations.jar file from the lgcbuild/ant/lib directory
  14. wait a long time for NetBeans to "scan" and compile all the source files
