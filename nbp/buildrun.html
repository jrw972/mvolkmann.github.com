Building Demo App.

The steps to build and run the "TabView Example 2" demo application under Windows are as follows:

  1. get the file Landmark-dsinfr-win32.xml and copy it to C:\Documents and Settings\{username}\.ivy2\cache\dsinfra-LOCAL
  2. open a Command Prompt
  3. cd to the dsinfra directory (mine is C:\Halliburton\infra)
  4. run "lgcbuild module"
  5. select release specification 3
  6. run "lgcbuild start"
  7. select release specification 3
  8. the "Decision Space" launcher will be displayed
  9. select "TabView Example 2"
