Decision Space (DS) Menus

The interfaces that manage DS menu bars, menus and menu items are AWMenuBar, AWMenu and AWMenuItem. The classes that implement those interfaces are CAWMenuBar, CAWMenu, CAWMenuItem.

The DS framework supports many types of menus. The following classes are identical except for their names. Their purpose is not clear.

  • BoxMenuImpl - menu for Box plots
  • HistogramMenuImpl - menu for Histogram plots
  • HorizontalTrimMenuImpl - menu for HorizontalTrim plots
  • PDFEditorMenuImpl - menu for the PDF editor
  • PolygonMenuImpl - menu for Polygon plots
  • TwoPointLineMenuImpl - menu for TwoPointLine plots
  • VerticalTrimMenuImpl - menu for VerticalTrim plots

The DS framework contains classes for managing specific menus. The following classes are identical except for their names. Their purpose is not clear.

  • ApplicationMenuImpl
  • DataSetMenuImpl
  • EditMenuImpl
  • HelpMenuImpl
  • ScriptingMenuImpl
  • WindowMenuImpl
